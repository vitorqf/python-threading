# Configurando o projeto

Crie um ambiente virtual python e instale as dependencias

```bash
# Linux
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# Windows
python -m venv .venv
.\venv\Scripts\activate
pip install -r requirements.txt
```

# Inicializando

```bash
# Linux
python3 main.py

# Windows
python main.py
```
